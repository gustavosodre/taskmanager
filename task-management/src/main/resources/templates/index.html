Task Manager

This application will help you get organized

Access to most features is restricted to logged-in users. Unauthorized users can only access the welcome screen and the login or registration panel.

Manager:

  • Create a task and assign the task to any user
  • View the list of all users
  • Delete users
  • View the list of all tasks
  • Edit and delete task

Employee:

  • Create personal task
  • View the list of all users
  • View list of all tasks
  • Edit and delete own tasks

Authorized users:

  • View own profile
  • Switch task as completed/uncompleted
